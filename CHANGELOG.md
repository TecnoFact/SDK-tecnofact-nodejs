# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of TecnoFact SDK for Node.js
- Support for CFDI 4.0 invoice creation
- Configuration management with Environment support (Sandbox/Production)
- HTTP client with automatic error handling
- Complete data models for Mexican electronic invoicing
- TypeScript support with full type definitions
- Custom exception hierarchy for better error handling
- Examples for basic usage and error handling
- Comprehensive documentation in README.md

### Features
- **Config**: Immutable configuration with validation
- **Enums**: Environment and TipoComprobante enumerations
- **Exceptions**: Custom exception classes for different error types
- **Models**: Complete data models (Emisor, Receptor, Concepto, etc.)
- **HttpClient**: Axios-based HTTP client with retry logic
- **TypeScript**: Full TypeScript support with strict typing

### Documentation
- Complete README with usage examples
- API documentation for all public methods
- Example files for common use cases
- Environment variable configuration guide
